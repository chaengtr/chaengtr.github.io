CSS Specificity

Selector มี Specificity Value ไม่เท่ากัน

เมื่อมี Selector มากกว่าหนึ่งทางเลือก ตัวที่ถูกเลือกใช้งานจะเป็นตัวที่มี Specificity Value มากที่สุด แต่ถ้ามี Specificity Value เท่ากัน จะเลือกตัวล่าสุดมาใช้งาน 

	ตัวอย่างที่ 1: Selector มีเพียงตัวเดียว (เลือก Selector นั้น)
			p1 {
			    color: red;
			}
This is my result. 
	ตัวอย่างที่ 2: Selector มีมากกว่าหนึ่งตัว (เลือกตัวที่ Specificity Value มากสุด)
			p2.active {
			    color: green;
			}
			p2 {
			    color: red;
			}
This is my result. 
	ตัวอย่างที่ 3: Selector ที่เหมือนกัน (เลือก Selector ตัวล่าสุด)
			p3.active {
			    color: green;
			}
			p3.active {
			    color: blue;
			}
This is my result.
